Etkin 3d edition typo's

Boeing 747 mathematical model

There are some typographical errors in Etkin's B747 model on p.166, 3d. edition. They may not be essential, but they can bite you if you try to use his models for verifying your own thinking.

The Imperial ( ft, lbf ) values in Table 6.2 are consistent with his eigenvalues and eigenvectors.

However, the SI part of the table has an exponent misprint in Mw. The value should be :

  Mw = -1.563e5

Another peculiar error is in the value of Czû in Table 6.1. The value matching Table 6.1 would be ****.
This is also much more in line with p.****, which shows that from physical reasoning, the non-dimensional value should be around - 2 CL, and p.165 states that :

  CLo = 0.654

lists of symbols

Etkin has the unfortunate habit of ending every chapter with an incremental list of symbols.
This makes his notation sometimes hard to track.

non-dimensional time

The time derivative of α  in Table 6.1 is made non-dimensional by an extra term of (c / 2V) relative to α •. This is a leftover from previous editions, and it is very poorly documented in the third edition.

  For the eagle eyed, there is a caret on top of the dot in the α • of Table 6.1. This caret seems to indicate "deriv­ative with respect to a non-dimensional time t̂ ". This non-dimensional time can be reconstructed with some effort from equations (5.5,1) and (5.5,5).
